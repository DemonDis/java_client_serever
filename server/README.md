# CLIENT (java)

## Server start
```bash
# install node_modules
npm i
# run server https
node app_https.js
# run server websocket
node app_socket.js
# run server http
node app_http.js
```

### Technology stack
1. NODE - **v21.7.1**
2. JAVA - **"17.0.12" 2024-07-16 LTS**
